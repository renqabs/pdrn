如遇到报错429,在setting下，拉到最下方，点击 Suspend Web Service（等同暂停），等待三分钟以上，回到setting下点击Resume Web Service（等同启动），等待1分钟左右，出现版本信息以及[GIN]开头的日志就表明pandora运行成功（即使状态显示为Failed deploy 也可以正常使用）
